## 🚀 Deployment Instructions

### Quick Deploy (5 minutes)

1. **Push to GitHub**
   ```powershell
   cd C:\OOPS
   git add .
   git commit -m "Initial: Java Learning Hub with complete documentation"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Navigate to Pages (left sidebar)
   - Select "Deploy from a branch"
   - Choose: Branch = `main`, Folder = `/docs`
   - Click Save

3. **Wait for Deployment**
   - GitHub will deploy automatically (1-2 minutes)
   - Your site will be live at: `https://yourusername.github.io/OOPS/`

### Automatic Deployments with GitHub Actions

The repository includes a GitHub Actions workflow (`.github/workflows/deploy.yml`) that automatically deploys whenever you push to `main`. No additional setup needed!

**See `docs/DEPLOYMENT.md` for detailed instructions.**
