Introduction to AI in Java

Artificial Intelligence is transforming how applications work. Java offers powerful frameworks to build AI-powered applications:

Key Concepts:
  • ✓ LLMs (Large Language Models): GPT-4, Claude, Llama, etc.
  • ✓ RAG (Retrieval-Augmented Generation): Combine docs with LLM responses
  • ✓ Agents: AI systems that can use tools and make decisions
  • ✓ Multi-Agent Systems: Multiple agents collaborating to solve problems
  • ✓ Embeddings: Convert text to vectors for semantic search

Why Java for AI?

  • ✓ Enterprise-grade frameworks (Spring Boot, Spring AI)
  • ✓ Strong type system and error handling
  • ✓ Rich ecosystem (LangChain4j, Agentscope)
  • ✓ Integration with existing backends
  • ✓ High performance and scalability

Spring AI Introduction

Overview

Spring AI provides a unified API for integrating AI/ML models into Spring Boot applications. It abstracts away provider-specific implementations (OpenAI, Anthropic, Ollama, etc.).

Setup Spring AI Project

// pom.xml - Add dependencies
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-openai-spring-boot-starter</artifactId>
    <version>0.8.0</version>
</dependency>

// For Ollama (local models)
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-ollama-spring-boot-starter</artifactId>
    <version>0.8.0</version>
</dependency>

Configure API Keys

// application.properties
spring.ai.openai.api-key=${OPENAI_API_KEY}
spring.ai.openai.chat.options.model=gpt-4o-mini

// For Ollama (local, no key needed)
spring.ai.ollama.base-url=http://localhost:11434
spring.ai.ollama.chat.options.model=llama2

Simple Chat Example

import org.springframework.ai.chat.ChatClient;
import org.springframework.ai.chat.messages.UserMessage;
import org.springframework.ai.chat.prompt.Prompt;
import org.springframework.stereotype.Service;

@Service
public class AIChatService {
    private final ChatClient chatClient;

    public AIChatService(ChatClient chatClient) {
        this.chatClient = chatClient;
    }

    public String chat(String userInput) {
        Prompt prompt = new Prompt(new UserMessage(userInput));
        return chatClient.call(prompt).getResult().getOutput().getContent();
    }
}

LangChain4j Complete Guide

What is LangChain4j?

LangChain4j brings the power of LangChain (Python) to Java. It provides abstractions for:

  • Language model interactions
  • Memory management
  • Chains and agents
  • Tool integration
  • RAG systems

Installation

// pom.xml
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j</artifactId>
    <version>0.28.0</version>
</dependency>

// For OpenAI
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-open-ai</artifactId>
    <version>0.28.0</version>
</dependency>

Basic Chat Example

import dev.langchain4j.model.chat.ChatLanguageModel;

Chat with Messages & Memory

import dev.langchain4j.model.chat.ChatLanguageModel;
import dev.langchain4j.model.openai.OpenAiChatModel;
import dev.langchain4j.model.chat.request.ChatRequest;
import dev.langchain4j.model.chat.request.ChatRequestParameters;
import dev.langchain4j.model.chat.response.ChatResponse;
import dev.langchain4j.model.message.ChatMessage;
import dev.langchain4j.model.message.UserMessage;
import dev.langchain4j.model.message.SystemMessage;
import java.util.*;

public class ChatWithMemory {
    public static void main(String[] args) {
        ChatLanguageModel model = OpenAiChatModel.builder()
            .apiKey(System.getenv("OPENAI_API_KEY"))
            .modelName("gpt-4o-mini")
            .build();

        List<ChatMessage> messages = new ArrayList<>();

        // Add system context
        messages.add(new SystemMessage("You are a helpful Java assistant."));

        // User message 1
        messages.add(new UserMessage("What is a HashMap?"));
        ChatResponse response1 = model.generate(messages);
        String answer1 = response1.content().text();
        System.out.println("Q: What is a HashMap?");
        System.out.println("A: " + answer1);

        // Add assistant response to memory
        messages.add(new ChatMessage.AssistantMessage(answer1));

        // User message 2 (with context)
        messages.add(new UserMessage("How does it work internally?"));
        ChatResponse response2 = model.generate(messages);
        System.out.println("\nQ: How does it work internally?");
        System.out.println("A: " + response2.content().text());
    }
}

RAG (Retrieval-Augmented Generation)

What is RAG?

RAG combines document retrieval with generation. Instead of relying solely on the model's training data, it:

  1. Takes user query
  2. Searches relevant documents
  3. Combines documents with query
  4. Generates response based on retrieved documents

RAG with LangChain4j

Step 1: Create Document Store

import dev.langchain4j.data.document.Document;
import dev.langchain4j.data.document.loader.FileSystemDocumentLoader;
import dev.langchain4j.data.message.UserMessage;
import dev.langchain4j.model.embedding.EmbeddingModel;

// Load documents from files
List<Document> documents = FileSystemDocumentLoader.loadDocuments(
    Paths.get("src/main/resources/docs"),
    doc -> true
);

System.out.println("Loaded " + documents.size() + " documents");

Step 2: Create Embeddings & Vector Store

import dev.langchain4j.data.embedding.Embedding;
import dev.langchain4j.model.embedding.EmbeddingModel;
import dev.langchain4j.model.openai.OpenAiEmbeddingModel;
import dev.langchain4j.store.embedding.InMemoryEmbeddingStore;

// Create embedding model
EmbeddingModel embeddingModel = new OpenAiEmbeddingModel.Builder()
    .apiKey(System.getenv("OPENAI_API_KEY"))
    .build();

// Create in-memory vector store
InMemoryEmbeddingStore embeddingStore = new InMemoryEmbeddingStore();

// Index documents
for (Document doc : documents) {
    Embedding embedding = embeddingModel.embed(doc.text()).content();
    embeddingStore.add(embedding, new TextSegment(doc.text(), doc.metadata()));
}

Step 3: Build RAG Chain

import dev.langchain4j.rag.RetrievalAugmentor;
import dev.langchain4j.rag.content.retriever.ContentRetriever;
import dev.langchain4j.rag.content.retriever.EmbeddingStoreContentRetriever;

// Create content retriever
ContentRetriever contentRetriever = new EmbeddingStoreContentRetriever(
    embeddingModel,
    embeddingStore,
    3  // Top-k results
);

// Build RAG chain
String userQuestion = "What is Java?";
RetrievalAugmentor ragAugmentor = new DefaultRetrievalAugmentor();

// Retrieve relevant documents
List<Content> retrievedContent = contentRetriever.retrieve(
    new UserMessage(userQuestion)
);

// Generate response based on retrieved content
String response = model.generate(
    buildPromptWithContent(userQuestion, retrievedContent)
);

System.out.println("Response: " + response);

Complete RAG Example

public class RAGExample {
    public static void main(String[] args) {
        // 1. Create sample documents
        List<String> documents = Arrays.asList(
            "Java is a high-level, class-based object-oriented language.",
            "Python is known for its simplicity and readability.",
            "Spring Boot is a framework for building Java applications.",
            "Machine learning enables computers to learn from data."
        );

        // 2. Create embedding model
        EmbeddingModel embeddingModel = new OpenAiEmbeddingModel.Builder()
            .apiKey(System.getenv("OPENAI_API_KEY"))
            .build();

        // 3. Index documents
        InMemoryEmbeddingStore store = new InMemoryEmbeddingStore();
        for (String doc : documents) {
            Embedding embedding = embeddingModel.embed(doc).content();
            store.add(embedding, new TextSegment(doc, null));
        }

        // 4. Retrieve similar documents
        String query = "Tell me about Java";
        Embedding queryEmbedding = embeddingModel.embed(query).content();
        List<EmbeddingMatch<TextSegment>> matches =
            store.findRelevant(queryEmbedding, 2);

        // 5. Generate response
        ChatLanguageModel model = new OpenAiChatModel.Builder()
            .apiKey(System.getenv("OPENAI_API_KEY"))
            .build();

        String context = matches.stream()
            .map(m -> m.embedded().text())
            .collect(Collectors.joining("\n"));

        String prompt = "Based on: " + context + "\n\nAnswer: " + query;
        String response = model.generate(prompt);

        System.out.println("Answer: " + response);
    }
}

Agents and Tool Use

What are Agents?

An agent is an AI system that can:

  • Take user input
  • Decide which tools to use
  • Execute tools with appropriate parameters
  • Process results and decide next steps
  • Iterate until problem is solved

Define Tools

import dev.langchain4j.agent.tool.Tool;

public class MathTools {

    @Tool("Adds two numbers")
    public int add(int a, int b) {
        return a + b;
    }

    @Tool("Subtracts b from a")
    public int subtract(int a, int b) {
        return a - b;
    }

    @Tool("Multiplies two numbers")
    public int multiply(int a, int b) {
        return a * b;
    }

    @Tool("Searches the web for information")
    public String search(String query) {
        // Simulate web search
        return "Search results for: " + query;
    }
}

Create Agent with Tools

import dev.langchain4j.agent.Agent;
import dev.langchain4j.agent.AgentBuilder;

public class AgentExample {
    public static void main(String[] args) {
        ChatLanguageModel model = new OpenAiChatModel.Builder()
            .apiKey(System.getenv("OPENAI_API_KEY"))
            .build();

        MathTools tools = new MathTools();

        Agent agent = new AgentBuilder()
            .chatLanguageModel(model)
            .tool(tools)  // Register all @Tool methods
            .build();

        // User query
        String response = agent.chat("What is 5 + 3 multiplied by 2?");
        System.out.println("Agent: " + response);

        // Another query
        String response2 = agent.chat("Search for latest Java news");
        System.out.println("Agent: " + response2);
    }
}

Agentscope Framework

Introduction to Agentscope

Agentscope is an open-source framework for building multi-agent systems. While primarily Python-based, it can be integrated with Java backends.

Key Features:
  • ✓ Distributed agent execution
  • ✓ Built-in MSGHub for agent communication
  • ✓ Service toolkit integration
  • ✓ Memory and state management
  • ✓ Web UI for monitoring

Java Integration with Agentscope

You can call Agentscope services from Java via REST API:

import okhttp3.*;

public class AgentScopeClient {
    private static final String AGENTSCOPE_URL = "http://localhost:8000";

    public static String callAgent(String agentName, String message)
            throws IOException {
        OkHttpClient client = new OkHttpClient();

        RequestBody body = RequestBody.create(
            "{\"agent\": \"" + agentName + "\", \"message\": \"" + message + "\"}",
            MediaType.get("application/json; charset=utf-8")
        );

        Request request = new Request.Builder()
            .url(AGENTSCOPE_URL + "/chat")
            .post(body)
            .build();

        try (Response response = client.newCall(request).execute()) {
            return response.body().string();
        }
    }

    public static void main(String[] args) throws IOException {
        String response = callAgent("assistant", "What is machine learning?");
        System.out.println("Response: " + response);
    }
}

Multi-Agent Architectures

Common Patterns

1. Coordinator Agent Pattern

One coordinator agent delegates tasks to specialized agents.

public class CoordinatorAgent {
    private RetrievalAgent retriever;
    private ExecutionAgent executor;
    private ReasoningAgent reasoner;

    public String handleUserRequest(String request) {
        // 1. Coordinator understands request
        String analysis = reasoner.analyze(request);

        // 2. Retriever gets relevant info
        String context = retriever.search(analysis);

        // 3. Executor performs actions
        String result = executor.execute(context);

        // 4. Return final response
        return reasoner.synthesize(request, result);
    }
}

2. Debate/Consensus Pattern

Multiple agents propose solutions and reach consensus.

public class ConsensusMultiAgent {
    private List<Agent> agents;

    public String solveByConsensus(String problem) {
        // Each agent proposes solution
        List<String> proposals = agents.stream()
            .map(agent -> agent.propose(problem))
            .collect(Collectors.toList());

        // Agents evaluate proposals
        Map<String, Integer> votes = new HashMap<>();
        for (Agent agent : agents) {
            String best = agent.evaluate(proposals);
            votes.merge(best, 1, Integer::sum);
        }

        // Return consensus solution
        return votes.entrySet().stream()
            .max(Map.Entry.comparingByValue())
            .map(Map.Entry::getKey)
            .orElse("No consensus");
    }
}

3. Hierarchical Planning

High-level planner breaks down tasks for specialized agents.

public class HierarchicalMultiAgent {
    private PlanningAgent planner;
    private List<SpecializedAgent> workers;

    public String executeGoal(String goal) {
        // 1. Planner creates task breakdown
        List<Task> tasks = planner.planTasks(goal);

        // 2. Assign tasks to specialized workers
        List<String> results = new ArrayList<>();
        for (Task task : tasks) {
            SpecializedAgent worker = selectBestWorker(task);
            result.add(worker.execute(task));
        }

        // 3. Synthesize results
        return planner.synthesizeResults(results);
    }

    private SpecializedAgent selectBestWorker(Task task) {
        return workers.stream()
            .max(Comparator.comparing(w -> w.expertise(task.type())))
            .get();
    }
}

Complete Code Examples

Example 1: Document Q&A System

public class DocumentQASystem {
    private ChatLanguageModel model;
    private InMemoryEmbeddingStore store;
    private EmbeddingModel embeddingModel;

    public void indexDocuments(List<String> docs) {
        for (String doc : docs) {
            Embedding emb = embeddingModel.embed(doc).content();
            store.add(emb, new TextSegment(doc, null));
        }
        System.out.println("Indexed " + docs.size() + " documents");
    }

    public String answerQuestion(String question) {
        // Retrieve similar docs
        Embedding qEmb = embeddingModel.embed(question).content();
        List<EmbeddingMatch<TextSegment>> matches =
            store.findRelevant(qEmb, 3);

        // Build context
        String context = matches.stream()
            .map(m -> m.embedded().text())
            .collect(Collectors.joining("\n---\n"));

        // Generate answer
        String prompt = "Context:\n" + context +
                       "\n\nQuestion: " + question +
                       "\n\nAnswer:";
        return model.generate(prompt);
    }
}

Example 2: Multi-Tool Agent

public class SmartAssistant {
    @Tool("Get current weather")
    public String getWeather(String city) {
        return "Weather in " + city + ": Sunny, 72Â°F";
    }

    @Tool("Calculate distance")
    public double calculateDistance(double lat1, double lon1,
                                    double lat2, double lon2) {
        // Haversine formula
        double R = 6371;
        double dLat = Math.toRadians(lat2 - lat1);
        double dLon = Math.toRadians(lon2 - lon1);
        double a = Math.sin(dLat/2) * Math.sin(dLat/2) +
                  Math.cos(Math.toRadians(lat1)) *
                  Math.cos(Math.toRadians(lat2)) *
                  Math.sin(dLon/2) * Math.sin(dLon/2);
        double c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
        return R * c;
    }

    @Tool("Get flight info")
    public String getFlights(String from, String to, String date) {
        return "Flights from " + from + " to " + to +
               " on " + date + ": BA123, UA456, DL789";
    }
}
